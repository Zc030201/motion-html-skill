High-end interactive HTML deck and 3D animation toolkit for AI training and demos.

## Highlights

- Cinematic deck layout with 3D interactive scenes
- Reusable effects (slide interactions, physics, particle/title unfold, 3D agents)
- Built for slide-like storytelling with click/arrow navigation
- Local preview ready for Windows and cross-platform

- [AI Training Deck (Slide 1~)](http://127.0.0.1:8791/ai-training-html-deck.html)
- [Bilibili 11-15 inspired module](http://127.0.0.1:8791/bili-11-15-replica.html)

## Quick start

```bash
./start-preview.ps1
```

Then open:

- `http://127.0.0.1:8791/ai-training-html-deck.html`
- `http://127.0.0.1:8791/bili-inspired-growth-module.html`

> If you are using `cmd`, use `start-preview.cmd` instead.

## Files

- `ai-training-html-deck.html` - Main 15+ page interactive deck
- `bili-11-15-replica.html` - 3D/physics-inspired visual module
- `bili-inspired-growth-module.html` - alternative standalone module
- `assets/` - Images and reusable visual resources
- `scripts/` - Optional demo/maintenance helpers

## Environment

- Windows 10/11
- PowerShell 5+/7
- Browser: Edge / Chrome / Firefox
